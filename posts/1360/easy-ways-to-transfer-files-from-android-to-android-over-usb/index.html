A Step by Step Guide on How to Connect Two Android Devices via USB

Transferring Files Between Android Devices Through USB Step by Step Guide to Transfer Files From Android to Android over USB

Step 1: Connect your devices Connect your devices to each other using a USB cable.

Step 2: Activate USB Debugging On the device you wish to receive files, ensure USB debugging is enabled. To do this, go to Settings > About Phone and tap Build Number seven times. Go back to Settings > Developer Options and make sure USB Debugging is toggled on.

Step 3: Accept the debugging connection By default, when two devices are connected, the receiving device will show a prompt to accept the connection. Tap OK to accept.

Step 4: Activate File Transfer Mode Once the connection is accepted, the senders device will display a prompt asking what type of connection you’d like to use. Tap File Transfer mode.

Step 5: Access File Sharing The receiving device will now display a popup with the contents of the senders device. Tap the file you’d like to receive, or select multiple files and hit Sync.

Step 6: Transfer Files The receiving device will now start downloading the files. Once the files are transferred, they will be accessible on the receiving device.

1. Preparing the Android Devices#

  1. Connect both Android devices to each other using a USB cable.
  2. Go to the receiving device’s Settings, then tap About Phone.
  3. Tap Build Number seven times to enable Developer Options.
  4. Go back to Settings, then tap Developer Options.
  5. Turn on the USB Debugging option.
  6. If you’re prompted with a pop-up window on the receiving device, tap OK to confirm the debugging connection.

2. Connecting the Devices with a USB Cable#

  1. Ensure both Android devices have a compatible USB cable, and then plug one end into the sender device and the other into the receiver device.
  2. On the receiver device, navigate to Settings > About Phone and tap the Build Number 7 times rapidly. This will enable Developer Options.
  3. Go to Settings > Developer Options and enable USB Debugging. This will allow both devices to communicate.
  4. The sender device will show the USB Debugging connection permission prompt at the top of the screen. Press OK to accept it.
  5. On the sender device, navigate to the files you want to transfer and tap the ‘Share’ icon. Select the receiver device from the list of options.
  6. Lastly, choose the storage location on the receiver device to which you would like to transfer the files and then tap the ‘Send’ option.

3. Allow USB File Transfer on Both Devices#

Thirdly, connect the receiving and sending Android devices with the USB cable. After connecting both the devices, allow USB transfer on either device to confirm the connection. For allowing USB transfer, go to Settings > More > USB utilities > and tap on Connect Storage to PC. It will provide you an option to allow USB file transfer and also enable USB debugging. Once these options are enabled, you can finally start transferring files from one Android device to another.

4. Transferring Files from One Android Device to Another#

  1. Both of the Android devices need to be connected with a USB cable.
  2. Once connected to each other, enter the Settings menu on the receiver device.
  3. Once there, select the About Phone option, and then tap Build Number seven times.
  4. This will unlock the Developer Options menu. Select it and turn on USB Debugging, so that the sender device can detect the receiving device.
  5. Once enabled, a prompt will appear on the sender device. Agree to the prompt and files will begin transferring from sender to receiver.
  6. When the process is finished, a notification will appear on both devices indicating that the transfer has been completed.
