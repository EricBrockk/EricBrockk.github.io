3. Steps to Updating the Android Version#

  1. Before attempting to update the Android version, ensure that the device has the necessary battery percentage to make it through the entire process. Connect it to a charger if needed.
  2. Open the Settings app and select the System option. Tap on ‘About phone’ and select System Update.
  3. The device will now check for any available updates to the Android version, and if found, will prompt you to install it. Tap ‘Yes’ to begin the installation process.
  4. The device will now download the version update files and install them automatically. During this time, do not power off the device or interfere with the process in any way.
  5. Once the installation process has been completed, the device will now reboot itself. This helps complete the update process.
  6. Once the device has rebooted, the Android version installed on it will be the latest one available. Check this by going into the ‘Settings’ menu and selecting the ‘About phone’ option.
