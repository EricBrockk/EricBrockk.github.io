1. Check Android version available for Redmi Note 6 Pro#

Step 1: On your Redmi Note 6 Pro, go to “Settings”, and then tap on “About Phone”.

Step 2: Under “Software Information”, you will find the current version of Android that your Redmi Note 6 Pro is running on.

Step 3: If you see the message “System is up to date” and the version is the one that you are using, then it means no further update is available.

Step 4: If an update is available, you will be asked whether you want to download and install the update. Tap on Download and install.

Step 5: Your Redmi Note 6 Pro will start to download the update. A progress bar will appear at the top of the screen.

Step 6: After the download is complete, the update will be installed automatically. Once the update is completed, your phone will restart to finish the installation process.

2. Download and install the update#

1.Back up important data, this step is necessary to keep them safe in case of any unexpected failure during the update process.

2.Navigate to Settings > About Phone > Software Information to check which Android version your phone is running and if an update is available.

3.If an update is available then download and install the update.

4.When the download is complete, tap Install Now to start the installation.

5.The phone will now reboot and start installing the update automatically. Make sure to keep the phone plugged in during the update.

6.Once the update is installed, your phone will reboot and you can use your device with the updated Android version.

3. Wait for the update to install#

1.In the first step, go to Settings > About Phone > System Update. 2.Tap on ‘Check for Updates’. 3.If an update is available, tap on ‘Download’ to start downloading the update. 4.Once the update is downloaded, tap on ‘Install Now’. 5.Your phone will now start installing the update and show the progress. 6.When the installation is complete, tap on ‘Reboot’ to restart the device and apply the update.

4. Check to make sure the update is successful#

  1. After the update is downloaded, the phone will prompt you to install it. Tap on the “Install Now” button to begin. Once the installation is complete, the phone will prompt you to restart your device to apply the update. Tap on the “Restart Now” button to finish the update.

  2. After rebooting, go back to the Software Information page to check if the update was successful. You should see the new version of Android you have installed. The installation process usually takes a few minutes, so be patient.

  3. After the update is successful, you might need to reinstall some of your apps. This is because the new update might not be compatible with the apps you have installed. To reinstall these apps, go to Google Play Store and search for the apps.

  4. It is also important to check for any system or app updates after updating your Android version. This is to make sure that your device is running the latest version of the operating system. To check for updated apps, go to Google Play Store and browse the available updates. Tap on the “Update All” button to update the apps if available.
