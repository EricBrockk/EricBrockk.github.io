3. Steps on how to update the Android version#

  1. Download the appropriate update package: From the official website, Redmi 9 users should download the latest version file onto the device.
  2. Verify the integrity of the package: The download package should be verified for authenticity and integrity to ensure there is no malicious software installed on the device.
  3. Select the update package: Once integrity is verified, select the downloaded package and follow instructions provided to install the update.
  4. Backup data: Before installing the update, users should ensure that they have a backup of their data including contacts, messages and other important information.
  5. Update the ROM: When prompted, select Update to update Redmi 9’s OS to the latest version.
  6. After installation: Wait for a few minutes until the installation is complete and the device will automatically reboot. After reboot, users should check their device’s version to confirm that the update was successful.
